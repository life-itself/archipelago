# Archipelago

A curated map of places where a regenerative, second renaissance culture is seeding — intentional communities, retreat centres, festivals, and living experiments.

**Live site:** https://archipelago.lifeitself.org

## Structure

- `index.md` — home page (full-screen map + about section)
- `about.md` — FAQ and background
- `places/` — one markdown file per place
- `data/places.json` — generated JSON (do not hand-edit)
- `docs/` — contributor documentation
